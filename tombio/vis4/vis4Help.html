The Full taxon details visualisation (vis4)

This visualisation is a fast way to explore the taxonomic knowledge-base for any given species. It displays all information including character state values, images and text files side-by-side for a species selected from a drop-down list.

To use the visualisation, just select a taxon from the drop-down list. When you've done this, all information in the knowledge base for this taxon will be displayed, including the character state values, images (if any referenced) and any other information in referenced HTML files (if any referenced). An example is shown below.

You can also tailor the display to exclude any of these three types of information (character state values, images or text) by unchecking the relevant check-boxes (also visible in the image below).
