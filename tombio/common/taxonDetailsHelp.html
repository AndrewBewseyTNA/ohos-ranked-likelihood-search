Taxon details dialog

If you click on the name of a taxon in the visualisation, a dialog box is shown which summarises all information and resources specified for the taxon in the knowledge-base.

There are three tabs on this dialog (illustrated side-by-side below) showing information on character states, images and supplementary (text) information. Each of these is described below.

Character state values

The character states specified against a taxon in the knowledge-base are shown on the first tab of this dialog.

Taxon images

The second tab of the dialog shows any images that are specified for the taxon in the knowledge-base. If more than one image is referenced for this taxon in the knowledge-base, you can change the image by clicking on one of the camera icons at the top of the image (there is one to represent each image). The camera icons become visible when you move your mouse cursor over the top of the image. Alternatively use the left and right arrows, visible when you move your mouse cursor to the left or right of the image, to move forward and backward through the image group. If you are using a touch device, turn image changing controls on/off by performing a press.

You can zoom in and out of an image by using your mouse wheel. When an image is zoomed in, you can pan it by dragging the image with your mouse. (Use pinch and drag gestures on a touch device to achieve the same result.)

Information (text) files

Knowledge-base authors can provide any number of text files (supplied as HTML files) to describe different aspects of any given taxa and link to these in their knowledge-base.

If only a single text file is referenced, it is simply displayed as supplied. If more than one text file is supplied, then they are listed in a drop-down list above the text display area and you can switch between them by selecting the one you want to see.
